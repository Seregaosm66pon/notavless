УСКОРИТЕЛЬ ИНТЕРНЕТА "Nota Vless"

Nota Vless - это проект, который был создан для обхода блокировок РКН, из за которых лично я потерял как минимум 1.5 тысячи рублей. Создан на удобство пользователей, и простоте. Добавляется за пару нажатий, и у вас есть постоянный доступ в интернет! Сам парсер имеет открытый исходный код на GitHub-е, если хотите что-то спросить по поводу проекта - пишите в личку канала, по более важным вопросам - лично овнеру, юз указан в био.
Огромный респект автору канала #РКП (https://t.me/rkp_channel), без его проекта не было бы и этого, его парсер тоже очень хорош, можете попробовать как альтернативу, если наш не работает.
Мы не кролики - мы люди.
